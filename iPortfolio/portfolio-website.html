Discription

During my tenure at IIT Madras Built-in Club, I immersed myself in a project that perfectly encapsulated the Research Park's ethos of innovation. This endeavor involved constructing a robust server infrastructure complemented by developing an interactive web application. My pivotal role spanned various project aspects. In server development, I contributed to designing, configuring, and optimizing the architecture, requiring a deep understanding of network protocols, security measures, and scalability. For the interactive web application, we crafted a user-friendly, feature-rich interface that seamlessly interfaced with the server, showcasing our commitment to user experience and front-end design while leveraging cutting-edge technologies for an engaging user experience.
